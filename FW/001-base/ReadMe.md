# 说明

此工程为`ESP-32S`的基础模板。

# 外设

## 烧录接口

烧录接口占用引脚如下:

| 引脚编号 | 功能  | 备注                           |
| -------- | ----- | ------------------------------ |
| GPIO1    | U0TXD | 烧录串口                       |
| GPIO3    | U0RXD | 烧录串口                       |
| GPIO0    |       | GPIO0与GPIO2配合可进入烧录模式 |
| GPIO2    |       | GPIO0与GPIO2配合可进入烧录模式 |

## 控制台

控制台将占用串口0(其同时也是烧录串口),其引脚占用如下:

| 引脚编号 | 功能  | 备注 |
| -------- | ----- | ---- |
| GPIO1    | U0TXD |      |
| GPIO3    | U0RXD |      |

## LED

LED引脚占用如下:

| LED编号 | 引脚编号 | 备注                                                       |
| ------- | -------- | ---------------------------------------------------------- |
| 0       | GPIO2    | 高电平驱动（保证上电瞬间处于低电平，不影响烧录模式的进入） |

# 编译

用户可在进入`esp-idf`环境后，使用`idf.py`进行编译。

注意:

- 必须使用指定版本的[esp-idf](../3rdparty/esp-idf),其它版本的esp-idf可能无法正常编译
