# 说明

 本目录为使用在zephyr使用hbox的基础应用。

# 外设

## 烧录接口

烧录接口占用引脚如下:

| 引脚编号 | 功能  | 备注                           |
| -------- | ----- | ------------------------------ |
| GPIO1    | U0TXD | 烧录串口                       |
| GPIO3    | U0RXD | 烧录串口                       |
| GPIO0    |       | GPIO0与GPIO2配合可进入烧录模式 |
| GPIO2    |       | GPIO0与GPIO2配合可进入烧录模式 |

## 控制台

控制台将占用串口0(其同时也是烧录串口),其引脚占用如下:

| 引脚编号 | 功能  | 备注 |
| -------- | ----- | ---- |
| GPIO1    | U0TXD |      |
| GPIO3    | U0RXD |      |

## LED

LED引脚占用如下:

| LED编号 | 引脚编号 | 备注                                                       |
| ------- | -------- | ---------------------------------------------------------- |
| 0       | GPIO2    | 高电平驱动（保证上电瞬间处于低电平，不影响烧录模式的进入） |
