# 说明

此工程为`ESP-32S`的基本以太网模板。

此工程依赖[001-base](../001-base).

# 外设

## 烧录接口

烧录接口占用引脚如下:

| 引脚编号 | 功能  | 备注                           |
| -------- | ----- | ------------------------------ |
| GPIO1    | U0TXD | 烧录串口                       |
| GPIO3    | U0RXD | 烧录串口                       |
| GPIO0    |       | GPIO0与GPIO2配合可进入烧录模式 |
| GPIO2    |       | GPIO0与GPIO2配合可进入烧录模式 |

## 控制台

控制台将占用串口0(其同时也是烧录串口),其引脚占用如下:

| 引脚编号 | 功能  | 备注 |
| -------- | ----- | ---- |
| GPIO1    | U0TXD |      |
| GPIO3    | U0RXD |      |

## LED

LED引脚占用如下:

| LED编号 | 引脚编号 | 备注                                                       |
| ------- | -------- | ---------------------------------------------------------- |
| 0       | GPIO2    | 高电平驱动（保证上电瞬间处于低电平，不影响烧录模式的进入） |

## 以太网

| 引脚编号 | 功能      | 备注                                                         |
| -------- | --------- | ------------------------------------------------------------ |
| GPIO0    | ETH_REFC  | 默认情况下，接以太网PHY芯片的时钟输出，由于GPIO0也用作       |
| GPIO5    | ETH_EN    | 接以太网PHY芯片的复位，注意:当ETH_EN为低时,以太网PHY芯片的时钟输出不应当与GPIO0连接。 |
| GPIO18   | ETH_MDIO  |                                                              |
| GPIO23   | ETH_MDC   |                                                              |
| GPIO19   | ETH_TX0   |                                                              |
| GPIO21   | ETH_TX_EN |                                                              |
| GPIO22   | ETH_TX1   |                                                              |
| GPIO25   | ETH_RX0   |                                                              |
| GPIO26   | ETH_RX1   |                                                              |
| GPIO27   | ETH_CRS   |                                                              |

ESP32支持以太网需要通过RMII接口连接以太网PHY芯片,默认情况下,本人使用LAN8720芯片作为以太网PHY芯片。

# 编译

用户可在进入`esp-idf`环境后，使用`idf.py`进行编译。

注意:

- 必须使用指定版本的[esp-idf](../3rdparty/esp-idf),其它版本的esp-idf可能无法正常编译
